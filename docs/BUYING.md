# Buying Layout Skill

Layout Skill is sold as a commercial source-available tool.

## Recommended Purchase Flow

1. Choose a license tier.
2. Complete payment through the configured purchase channel.
3. Send the GitHub username that should receive access.
4. Receive an invite to the private source repository.
5. Clone the repository and run `./install.sh`.
6. Install ODA File Converter and start processing DWG files locally.

## Suggested License Tiers

### Solo

For one independent consultant or planner.

Includes:

- one GitHub user access
- source repository access
- installation guide
- update access
- basic issue support

Suggested price: USD $99-$199 one-time, or USD $19-$29/month with updates.

### Studio

For small studios, agencies, or planning teams.

Includes:

- up to five GitHub users
- commercial client-work usage
- priority bug-fix support
- update access
- onboarding support

Suggested price: USD $499-$999 one-time, or USD $99-$199/month with updates.

### Enterprise

For larger organizations, white-label use, or custom deployment.

Includes:

- custom user count
- invoice support
- onboarding/training
- custom workflow integration
- optional white-label or hosted-service rights under separate agreement

Suggested price: quote-based.

## Purchase Channels

Use one of these:

- GitHub Sponsors tier with private repository access
- Stripe / Lemon Squeezy / Gumroad checkout
- direct invoice for studios and enterprise customers

## Access Management

Paid customers should provide:

- GitHub username
- license tier
- billing contact
- organization name, if applicable

Repository access should be removed when a subscription expires or when a
license is refunded.
